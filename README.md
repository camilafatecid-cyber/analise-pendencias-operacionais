# Análise de Pendências Operacionais

## 📌 Objetivo
Analisar o volume de pendências operacionais e identificar gargalos que impactam prazos e eficiência.

## 📊 Principais Insights
- 40% das tarefas estão atrasadas mesmo com tempo médio de resolução baixo (~3 dias)
- Concentração de pendências em usuários específicos
- Indício de gargalo na distribuição de tarefas

## 🛠 Ferramentas utilizadas
- Python (ETL e tratamento de dados)
- SQL
- Power BI
- Excel

## 🔄 Processo
1. Coleta de dados brutos
2. Tratamento e transformação (ETL)
3. Análise exploratória
4. Construção de dashboard
